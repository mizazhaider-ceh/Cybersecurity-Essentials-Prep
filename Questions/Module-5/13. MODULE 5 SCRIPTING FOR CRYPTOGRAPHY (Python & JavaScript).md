**Question 34** (üî¥ Hard - **CAESAR CIPHER!**)

**Caesar cipher Python implementation:**

```python
original_string = "HELLO"
modified_ascii_values = [(ord(char)-65+3)%26+65 for char in original_string]
modified_string = ''.join(chr(value) for value in modified_ascii_values)
print(modified_string)  # Output: ?
```

**Output is:**

A) HELLO

B) KHOOR

C) EBIIL

D) MJQQT

**Answer:** B

**Explanation:**
‚úÖ **KHOOR**

- H ‚Üí K (shift +3)
- E ‚Üí H
- L ‚Üí O
- L ‚Üí O
- O ‚Üí R

**How it works:**

- `ord(char)` converts char to ASCII (A=65, ..., Z=90)
- `-65` makes A=0, B=1, ..., Z=25
- `+3` shifts by 3
- `%26` wraps around (Z+3 ‚Üí C)
- `+65` converts back to ASCII
- `chr(value)` converts ASCII to char
