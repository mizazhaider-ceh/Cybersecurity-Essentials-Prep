## ğŸ¯ KEY EXAM STRATEGIES FROM EXERCISES

1. **Malware Can Have MULTIPLE Characteristics**
    - ILOVEYOU = Worm + Social Engineering + Phishing
    - WannaCry = Worm + Ransomware
    - Select ALL that apply!
2. **Defense in Depth Distinctions:**
    - **Concentric Rings** = nested layers, progression inward
    - **Overlapping Redundancy** = multiple controls at SAME location/layer
    - **Segregation** = EITHER multiple people needed OR network isolation
3. **Attack Vector vs. Malware Type:**
    - **Attack vector** = HOW it gets in (mail, network, USB)
    - **Malware type** = WHAT it is (virus, worm, trojan)
    - Don't confuse them!
4. **Log Analysis = Pattern Recognition:**
    - Multiple failed + 1 successful = Brute force
    - Massive requests from many IPs = DDoS
    - Related events from same IP = Coordinated attack
