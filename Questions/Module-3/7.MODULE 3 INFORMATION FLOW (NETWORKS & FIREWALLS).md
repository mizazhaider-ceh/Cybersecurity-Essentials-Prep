### **ğŸ“ What You MUST Know:**

1. **OSI 7 Layers** (mnemonic: "All People Seem To Need Data Processing"):
    - **A**pplication, **P**resentation, **S**ession, **T**ransport, **N**etwork, **D**ata Link, **P**hysical
2. **TCP/IP 4 Layers vs OSI**:
    - Application (OSI 7,6,5)
    - Transport (OSI 4)
    - Internet (OSI 3)
    - Network Interface (OSI 2,1)
3. **Firewall Generations**:
    - 1st: Packet Filtering (Layer 3, IP/Port)
    - 2nd: Stateful Inspection (tracks connections)
    - 3rd: Application (Layer 7, proxy, examines content)
    - Next Gen: DPI + Application awareness + IPS
4. **IDS vs IPS**:
    - IDS = **Detects** and **alerts**
    - IPS = **Detects** and **prevents/blocks**
5. **DMZ (Screened-subnet)**:
    - 2 firewalls creating zone between Internet and private network
    - Public servers (web server) go in DMZ
6. **DLP 3 States**:
    - Data at **rest** (crawler finds stored data)
    - Data in **transit** (DPI analyzes)
    - Data in **use** (checks permissions: print, USB, clipboard)
